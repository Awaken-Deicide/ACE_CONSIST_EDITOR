# ACE_CONSIST_EDITOR
A project for Glocol Networks to Fill up the Consist info.


**Requires Node.JS**
Download from: [NodeJS](https://nodejs.org/en/download)

**Requires VS-Code**
Download from: [VS-Code](https://code.visualstudio.com/)

## Steps to Run the Project
1. Clone the project or copy the Directory in your Local computer.
2. Open the project directory /ACE_Consist_Editor in VS-Code.
3. Run "npm i" in terminal to install all the dependencies.
4. Run "node app.js" to start the web app. A message should pop up stating "Server started or port 3000."
5. Go to http://localhost:3000/ in any of your web browser to access the WebPage.
